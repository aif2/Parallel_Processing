Parallel merge sort


This folder contains the following files:
README.txt
threaded_sort.cpp


the code is implemented using POSIX thread (pthread) libraries. Maximum size of array to sort is 1000000. Number of thread will be user defined. 

Compile and run the project using terminal as below:

	g++ threaded_sort.cpp -lpthread -o sort
	./sort

The above lines will compile the necessary files and run the program.
